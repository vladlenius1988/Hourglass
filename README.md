# Hourglass

Hourglass is a personal web organizer designed to help users plan tasks and goals according to their available time and energy.

The idea is simple: instead of just listing tasks, the system helps distribute them realistically throughout the day, making planning more intentional and balanced.

---

## 🎯 Who is it for?

- People who struggle with daily planning
- Users who want a structured yet flexible task system
- Anyone who wants to manage time more consciously
